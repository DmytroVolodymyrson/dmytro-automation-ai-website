# 2026-03-28

- Saved PostHog personal API key to local secrets store for direct API reads.
- Queried PostHog project `344218` directly to review Google Ads impact.
- Key finding: Google Ads is increasing traffic volume, but paid sessions are not converting yet on-site.
- Evidence from PostHog:
  - 2026-03-27 had 80 pageviews across 38 sessions, vs ~14 pageviews/day average across 2026-03-15 to 2026-03-26.
  - 2026-03-27 to 2026-03-28 had 56 sessions total; 47 sessions contained ad params (`gclid` / `wbraid` / `gbraid` / `gad_*`) in the landing URL.
  - Those 47 ad-attributed sessions produced 0 CTA sessions and 0 booking-widget-view sessions.
  - Clean/non-ad sessions in the same period produced 1 CTA session and 2 booking-widget-view sessions.
- Important caveat: `booking_widget_viewed` is a scroll/view proxy, not a booking completion event.
- Tracking gap found: PostHog queryable `utm_source` / `utm_medium` are null, so paid attribution currently has to be inferred from landing URL parameters. Attribution instrumentation should be tightened.
