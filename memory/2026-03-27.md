## GEO automation research
- Dmytro asked how to automate testing for ChatGPT, Claude, Gemini, and similar surfaces.
- Perplexity-backed research indicates the practical architecture is hybrid:
  - APIs for recurring automated sweeps
  - browser/UI spot checks for true consumer-product validation
- Current best official API path appears to be:
  - OpenAI Responses API with web search for ChatGPT-like grounded testing
  - Anthropic Claude API with web search
  - Gemini API with Google Search grounding
  - Perplexity API / Sonar for search-grounded answers
- Key caveat: API results may differ from consumer UI results, especially on ChatGPT Search, so API automation should be treated as trend tracking rather than perfect UI truth.
- Preferred future system: weekly scheduled prompt runner + stored outputs/citations/scores + occasional manual/browser validation.
